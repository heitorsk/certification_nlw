API REST
    Método comuns
- GET - Buscar uma informação
- PUT - Alterar uma informação
- POST - Inserir uma informação
- DELETE - Remover uma informação
- PATCH - Alterar PONTUAL de uma informação

Os tipos de parâmetros
- Body - {}
- Query Params - http://localhost:8080/users?nome=Heitor&idade=20 OPCIONAIS
- Parâmetros de rotas http://localhost:8080/users/{nome}/{idade}

ORM -> Object Relational Mapper

Repositório (Repository) -> Interagir com o banco de dados.
